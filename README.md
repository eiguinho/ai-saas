# Plataforma SaaS com Integração de Inteligência Artificial

Plataforma SaaS desenvolvida para geração de conteúdo com Inteligência Artificial, com foco em arquitetura escalável, organização modular e evolução contínua do produto.

O sistema integra diferentes modelos de linguagem para geração de texto e conteúdo multimodal, oferecendo autenticação de usuários, gerenciamento de planos, histórico de conversas, upload de arquivos e funcionalidades administrativas.

## Principais Funcionalidades

- Autenticação e gerenciamento de usuários  
- Geração de conteúdo com múltiplos provedores de IA  
- Histórico de conversas e conteúdos gerados  
- Painel administrativo  
- Controle de planos e permissões de acesso  
- Upload e gerenciamento de arquivos  
- Fluxos de geração multimodal
